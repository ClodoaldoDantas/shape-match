# 🧩 Shape Match

Shape Match é um jogo educativo simples para crianças, no qual o objetivo é arrastar formas geométricas e encaixá-las em seus contornos correspondentes.

![image](https://github.com/user-attachments/assets/20ad0ad9-ec68-43f3-9f9b-9f9efa04c518)

## ✨ Funcionalidades

- 🎮 Interação por arrastar e soltar (drag and drop)
- ✅ Verificação automática de acerto
- 💡 Design simples e intuitivo para o público infantil

## 👶 Público-alvo

O jogo foi criado com foco em crianças em fase de alfabetização ou educação infantil, promovendo o aprendizado visual e motor de forma divertida.

## 🔥 Instalação

1. Clone o repositório
2. Entre no diretório do projeto
3. Instale as dependências com `npm install`
4. Execute o comando `npm run dev` para iniciar o servidor
